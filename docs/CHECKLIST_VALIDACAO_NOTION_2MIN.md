# Checklist de Validação Final (2 minutos) — Notion Zero Base

## 1) Estrutura (30s)
- [ ] A página principal está com nome e identidade **Zero Base**.
- [ ] As 8 páginas principais estão visíveis e organizadas.
- [ ] Não há página duplicada com nome antigo.

## 2) Conteúdo profissional (30s)
- [ ] Cada página tem resumo curto no topo.
- [ ] Métricas estão com contexto/fonte (sem números soltos).
- [ ] Linguagem está objetiva (sem textos promocionais excessivos).

## 3) Database Roadmap (30s)
- [ ] Campos existem: Fase, Categoria, Prioridade, Status, Responsável, Prazo, Impacto.
- [ ] Existem views: Kanban (Status), Timeline (Prazo), Tabela (Prioridade).
- [ ] Tarefas críticas (P0/P1) estão priorizadas.

## 4) Evidências e links (20s)
- [ ] Link do GitHub funciona.
- [ ] Link da demo (ou vídeo) funciona.
- [ ] Screenshots principais foram adicionados (dashboard/timer/relatórios).

## 5) Qualidade de apresentação (10s)
- [ ] Ortografia revisada.
- [ ] Capa e ícone coerentes com o projeto.
- [ ] Página de apresentação acadêmica pronta para uso (roteiro 10 min).

---

## Critério de pronto
Se **todos os itens** estiverem marcados, sua documentação está pronta para:
1. apresentação ao professor;
2. uso como portfólio;
3. acompanhamento de execução por roadmap.
