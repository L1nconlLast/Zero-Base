# Product Core

## Objetivo
Zero Base é um app de estudos orientado a execução com quatro loops centrais:

1. Planejar
2. Executar
3. Revisar
4. Medir

## Fluxo principal
Onboarding  
→ Geração do plano  
→ Home  
→ Estudos  
→ Finalização de sessão  
→ Revisão 24h  
→ Atualização do progresso  
→ Desempenho

## Núcleo funcional
- Home
- Plano
- Estudos
- Revisões
- Desempenho
- Simulados
- Edital
- Perfil

## Regra de UX
- Sempre mostrar o próximo passo
- Sempre permitir começar com 1 clique
- Nunca esconder tarefas críticas
- Toda métrica mostrada deve derivar do estado central
