# FAQ - Perguntas Frequentes

## Instalação

**Como iniciar o projeto?**

```bash
npm install
npm run dev
```

Abra o endereço exibido no terminal.

## Build e testes

**Como validar rapidamente?**

```bash
npm run test -- --run
npm run build
```

## Autenticação

**Meu login não funciona. O que verificar?**

- E-mail e senha corretos.
- Conta criada com sucesso.
- Console do navegador sem erros críticos.

## Dashboard

**Os gráficos não aparecem.**

- Verifique se existem sessões registradas.
- Recarregue a página.
- Confirme se não há erro de importação no console.

## Dados

**Como fazer backup?**

Use a área de dados/configurações para exportar JSON e guardar localmente.

## PWA

**Como instalar como app?**

No Chrome/Edge, use a opção de instalar aplicativo no menu do navegador.

## Problemas comuns

**Porta ocupada**

```bash
npm run dev -- --port 3000
```

**Dependências quebradas**

```bash
npm install
```

## Suporte

Se o problema persistir, abra uma issue com passos para reprodução e logs relevantes.
