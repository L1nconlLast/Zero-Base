# Quick Start - Zero Base 2.0

## Começar em 3 Passos

### 1) Instalar dependências
```bash
npm install
```

### 2) Rodar o projeto
```bash
npm run dev
```

### 3) Acessar
Abra no navegador: `http://localhost:5173`

---

## Primeiro Uso

1. **Criar conta**
   - Clique em "Cadastre-se"
   - Preencha nome, e-mail e senha

2. **Fazer login**
   - Use o e-mail e senha cadastrados

3. **Começar a estudar**
   - Abra o Cronômetro
   - Clique em Iniciar
   - Finalize a sessão ao terminar

4. **Ver progresso**
   - Dashboard mostra gráficos e estatísticas
   - Conquistas mostra achievements desbloqueados

---

## Comandos Úteis

```bash
npm run dev      # Desenvolvimento
npm run build    # Build otimizada
npm run preview  # Preview da build
npm run lint     # Verifica código
```

## Problemas Comuns

### Erro ao instalar
```bash
npm install
```

### Página em branco
- Abra o console do navegador
- Verifique se há erro de dependência

### Gráficos não aparecem
```bash
npm list recharts
```
