# Contribuindo para Zero Base

Obrigado por contribuir.

## Fluxo sugerido

1. Faça um fork e crie uma branch.
2. Implemente mudanças pequenas e objetivas.
3. Rode testes e lint.
4. Abra pull request com descrição clara.

## Padrão de commits

- `feat:` nova funcionalidade
- `fix:` correção de bug
- `docs:` documentação
- `refactor:` refatoração sem mudança funcional
- `test:` testes
- `chore:` manutenção

## Checklist para PR

- [ ] Código compila
- [ ] Testes passam
- [ ] Lint sem erros
- [ ] Documentação atualizada quando necessário
