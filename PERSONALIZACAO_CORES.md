# Guia de Personalização - Cores e Temas

## Como mudar as cores

### Opção 1: Seletor de temas

1. Abra o aplicativo.
2. Vá para Configurações.
3. Escolha um tema disponível.
4. A preferência será salva automaticamente.

### Opção 2: Tailwind

Edite as cores em `tailwind.config.js` e reinicie o servidor:

```bash
npm run dev
```

## Dicas

- Teste contraste em modo claro e escuro.
- Evite combinações com baixa legibilidade.
- Valide aparência em telas menores.

## Aplicar mudanças

Depois de alterar tema ou configuração, recarregue a página para confirmar os ajustes visuais.
